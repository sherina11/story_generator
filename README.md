# Random Story Generator (Python)

This is a simple and fun Python project that generates a random story every time the program runs.

The program selects random characters, places, actions, and story endings to create a unique story.

## Features

* Generates a different story each time
* Uses Python random module
* Beginner-friendly project
* Easy to run

## Example Output

🌟 Your Random Story 🌟

A young programmer inside a futuristic city discovered a secret code and a new adventure started.

## Technologies Used

* Python

## How to Run

1. Install Python on your computer
2. Download or clone this repository
3. Open terminal in the project folder
4. Run the program

```
python story_generator.py
```
